#CAMARA

La camara se mueve usando:

w para alejar

a izquierda 

s acercar 

d derecha 

q abajo 

e arriba 

Se rota con:

z axis y positivo

x axis y negativo

c axis x positivo

b axis x negativo

v axis z positivo

n axis z negativo

Se utiliza r para volver a la posicion inicial

Se mueve la luz junto con la camara
